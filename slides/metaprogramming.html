Metaprogramming

  • Julia supports several types:
    • Int
    • Float
    • String
    • Expr
    • etc...
  • Expr is a representation of Julia code

    (homoiconicity)

  • Julia provides methods to manipulate Expr data
  • Julia can generate and run its own source code
  • At compile time:
    • by means of macros
    • Julia acts as its own preprocessor
  • At run time:
    • by means of eval()
    • (like other interpreted languages)
