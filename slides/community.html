Julia community

Small but very active: computer scientists, engineers, scientists, many enthusiastic persons involved in all kinds of numerical computing

  • The discourse: with $\sim$29.7k topics, $\sim$260k posts, $\sim$12k registered users, the best place to search for detailed informations and ask questions (stats);
  • The Zulip chat: with $\sim$100k messages since Apr. 2020, $\sim$1.5k daily active users (stats);
  • Github repository: core language development, with $\sim$1k contributors;
  • Julia ecosystem (contributed package repositories): $\sim$5k registered packages;
  • JuliaCon: annual Julia conference;
  • Youtube channel: $\sim$52k subscribers,all JuliaCon videos available;
