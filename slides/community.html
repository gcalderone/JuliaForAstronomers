Julia community

Small but very active: computer scientists, engineers, scientists, many enthusiastic persons involved in all kinds of numerical computing

  • The discourse: with $\sim$56k topics, $\sim$484k posts, $\sim$20k registered users, the best place to search for detailed informations and ask questions (stats);
  • The Zulip chat: with $\sim$428k messages (stats);
  • Github repository: core language development, with $\sim$1.4k contributors;
  • Julia packages (contributed package repositories): $\sim$9.9k registered packages;
  • JuliaCon: annual Julia conference;
  • Youtube channel: all JuliaCon videos available;
