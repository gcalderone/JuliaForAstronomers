Julia syntax

  • Julia syntax is straightforward: if you know any modern language, you will found Julia very easy to use;
  • Developing good libraries and macros is slightly more difficult;

  • Noteworthy differences from other languages:
    • Julia has an end block delimiter: end;
    • Indexing of arrays is 1-based. Last index is end;
    • Julia has no vectorized versions of functions (not needed for performance). To vectorize an expression use the dot syntax: sin.([0, pi/2])
    • By convention, function names terminating with ! modifies their inputs.
