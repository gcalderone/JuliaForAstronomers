The Gnuplot package

https://github.com/gcalderone/Gnuplot.jl/
  • Based on Gnuplot;
  • Extremely concise syntax;
  • Support for multiplots;
  • Exploit all 2D/3D Gnuplot functionalities as well as all terminals;
  • Save plots into gnuplot scripts;
 

x = linspace(-2pi, 2pi, 100);
y = 1.5 * sin.(0.3 + 0.7x) ;
noise = randn(length(x))./2;
e = 0.5 * ones(x);

using Gnuplot
@gp(tit="My title", xr=(-7,7), xlab="X label", ylabel="Y label",
    x, y, "w l t 'Model' dt 2 lw 2 lc rgb 'red'",
    x, y+noise, e, "w errorbars t 'Data'",
    term=("png", "gnuplot.png"))
