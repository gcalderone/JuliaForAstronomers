Multiple dispatch

  • Single dispatch $\equiv$ OOP
  • C++, Java, Python, etc...
  • Methods belong to data
  • Multiple dispatch
  • Julia
  • Methods and data are treated independently

Grammar analogy

  • Focus on subjects ($\equiv$ class)
  • All verbs ($\equiv$ methods) can only be applied to a single subject, e.g.
    • a person walks;
    • a socket sends a data packet;
    • a number is summed with another number
    • etc...
  • Focus on both subjects and actions
  • All verbs ($\equiv$ methods) can be applied to any number of subjects, e.g.
    • walk(person);
    • send(socket, data);
    • add(number, number);
    • add(number, number, number);
    • etc...
