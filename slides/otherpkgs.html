Other noteworthy packages

  • Astronomy related:
    • Astrolib: collection of generic astronomical facilities, translated from the IDL AstroLib procedure colection;
    • Cosmology: calculation of cosmological related quantities;
    • DustExtinction: tools for interstellar dust extinction;
    • WCS: World Coordinate System library for Julia (wrapper to C library);
    • Healpix: Julia functions that implement the Healpix spherical projection.
  • Other packages:
    • DataFrames: tools for working with tabular data in Julia;
    • MySQL: access databases through the MariaDB C connector library;
    • DBInterface: unified database interface for Julia;
    • Revise: automatically recompile functions (avoid re-starting Julia sessions)
