The two language problem

  • Different optimal tools for different tasks:
    • statistics: $\Rightarrow$ R;
    • linear algebra: $\Rightarrow$ Matlab;
    • string processing: $\Rightarrow$ Perl;
    • general programming: $\Rightarrow$ Python, Java;
    • performance critical: $\Rightarrow$ C, C++, Fortran;

On Python, this lead to significant improvements, but with a plethora of different approaches, each with pros and cons.

  • Julia library is (largely) written in Julia itself;
  • $\Rightarrow$ both development and execution are intrinsically faster;
  • Develop complete solutions in one language;
  • If needed, existing C and Python libraries can be used within Julia;
