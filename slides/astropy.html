PyCall

https://github.com/JuliaPy/PyCall.jl
  • Easy call function from all Python modules
  • Pass data to/from Python, with automatic type conversion
  • Share some data structure (e.g. arrays) without copyying them

Astropy

http://astropy.org, Ref. paper
  • Collection of Python modules providing several astronomy-related functionalities:
    • Units conversions
    • WCS support
    • FITS, HDF and VO support
    • Cosmological model
    • Photometry tools
    • online astronomical database querying
    • etc...
PyCall allows to exploit all astropy functionalities from Julia
