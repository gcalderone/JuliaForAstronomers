Julia code niftyness

  • Colors in the REPL, and thousands separator:
    printstyled("Julia has more than ", 5_000, " users!", color=:light_red, bold=true)
  • Use LaTeX symbols in code:
    \eulerTAB^(-\piTAB * im) \approxTAB -1
  • Rename a function:
    \eulerTAB^(-\piTAB * im) \approxTAB -1
  • Online help (Markdown) by prepending ?:
    julia> ?println
  • Package OhMyREPL (REPL syntax highlighting):
    julia> ?println
