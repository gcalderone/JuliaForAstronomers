The DataFitting package

https://github.com/gcalderone/DataFitting.jl
  • General purpose data-driven model fitting framework;
  • Handles data of any dimensionality;
  • Support for both Cartesian and Linear domains;
  • Model is built up by individual components (either built-in or user defined) combined with a mathematical expression;
  • all components results are cached $\Rightarrow$ performance boost!
  • Support for different minimizers (two are currently available);
